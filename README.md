Tab Closer
Extensão para Google Chrome que fecha todas as abas do navegador, mantendo apenas a aba ativa aberta. Ideal para quem costuma trabalhar com várias abas e deseja rapidamente limpar a janela do navegador.

Funcionalidades
Fecha todas as abas abertas, exceto a que está ativa.

Interface simples e intuitiva no popup.

Visual moderno e responsivo.

Comunicação eficiente entre popup e background (service worker).

Instalação manual (modo desenvolvedor)
Baixe ou clone este repositório.

Abra o Chrome e acesse: chrome://extensions

Ative o Modo desenvolvedor (canto superior direito).

Clique em Carregar sem compactação e selecione a pasta raiz do projeto.

O ícone da extensão estará disponível na barra de ferramentas.

Como usar
Clique no ícone da extensão Tab Closer no Chrome.

Na janela popup, clique no botão Fechar abas.

Todas as abas (exceto a atual) serão fechadas instantaneamente.

Uma mensagem informando quantas abas foram fechadas aparece no popup.
